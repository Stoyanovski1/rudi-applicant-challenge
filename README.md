# AngularChallenge

For the applicant:

- Clone this project
- Create a feature branch with your name/alias (whatever you prefer - so that we can associate the branch with you)
- Further information on what to do will be found when you start the dev server

## Development server

To start a local development server, run:

```bash
ng serve
```

Once the server is running, open your browser and navigate to `http://localhost:4200/`. The application will automatically reload whenever you modify any of the source files.
